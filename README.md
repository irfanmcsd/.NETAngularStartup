## 🛠 Core Technologies  
- **Frontend**: Angular 19.2 + NgRx State Management  
- **Backend**: .NET 9 WebAPI + EF Core 9  
- **Auth**: JWT + Cookie Authentication  
- **Database**: SQL Server/PostgreSQL + Redis Cache  
- **AI Optimization**: DeepSeek Code Analysis + GitHub Copilot
